Standard form tag

Simplest possible way of displaying a form, You can simply copy-paste this snippet and you will have a perfect form.

{% templatetag openblock %} load standard_form {% templatetag closeblock %}

<form action="." method="post" class="frm">
    {% templatetag openblock %} csrf_token {% templatetag closeblock %}
    <fieldset class="frm-horizontal">
        {% templatetag openblock %} standard_form form {% templatetag closeblock %}
        {% templatetag openblock %} standard_submit {% templatetag closeblock %}
    </fieldset>
</form>

 

Standard field tag

If you have to separate fields into different containers (because they have to be styled differently, for example), use the standard_field tag.

<form action="." method="post" class="frm">
    {% templatetag openblock %} csrf_token {% templatetag closeblock %}
    <fieldset class="frm-horizontal">
        <ol>
            <li>{% templatetag openblock %} standard_field form.field_name_one {% templatetag closeblock %}</li>
        </ol>
    </fieldset>
    <fieldset class="frm-vertical">
        <ol>
            <li>{% templatetag openblock %} standard_field form.field_name_two {% templatetag closeblock %}</li>
        </ol>
        {% templatetag openblock %} standard_submit {% templatetag closeblock %}
    </fieldset>
</form>

 

Standard widget tag

If you are doing something really fancy, you can break down a field even more – to the bare field widget. This example also includes it's own submit button.

<form action="." method="post" class="frm">
    {% templatetag openblock %} csrf_token {% templatetag closeblock %}
    <fieldset class="frm-horizontal">
        <ol>
             <li>
                 <label for="id_{{ form.field_name_one.name }}">My label or other stuff</label>
                 <div class="field">{% templatetag openblock %} standard_widget form.field_name_one {% templatetag closeblock %}</div>
             </li>
             <li>
                 <label class="empty"></label>
                 <div class="field"><input type="submit" class="btn" value="{% templatetag openblock %} trans 'Go' {% templatetag closeblock %}" /></div>
             </li>
        </ol>
    </fieldset>
</form>
